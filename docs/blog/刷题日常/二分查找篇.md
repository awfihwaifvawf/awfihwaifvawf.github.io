# 2026-4-11 leeCode-278
::: info 题目详情
[278. 第一个错误的版本](https://leetcode.cn/problems/first-bad-version/) 

你是产品经理，目前正在带领一个团队开发新的产品。不幸的是，你的产品的最新版本没有通过质量检测。由于每个版本都是基于之前的版本开发的，所以错误的版本之后的所有版本都是错的。

假设你有 `n` 个版本 `[1, 2, ..., n]`，你想找出导致之后所有版本出错的第一个错误的版本。

你可以通过调用 `bool isBadVersion(version)` 接口来判断版本号 `version` 是否在单元测试中出错。实现一个函数来查找第一个错误的版本。你应该尽量减少对调用 API 的次数。

**示例 1：**

```
输入：n = 5, bad = 4
输出：4
解释：
调用 isBadVersion(3) -> false 
调用 isBadVersion(5) -> true 
调用 isBadVersion(4) -> true
所以，4 是第一个错误的版本。
```

**示例 2：**

```
输入：n = 1, bad = 1
输出：1
```

**提示：**

- `1 <= bad <= n <= 231 - 1`
:::

::: details 个人思路
假如第一个选取到的版本结果为true,如果前一个版本也为false,那么当前版本就是我们要查找的版本,如果前一个版本为ture，那么我们继续向前二分查找

假如第一个选取到的版本结果为false,如果后一个版本也为true,那么当前版本就是我们要查找的版本,如果后一个版本为false，那么我们继续向后二分查找

``` java
/* The isBadVersion API is defined in the parent class VersionControl.
      boolean isBadVersion(int version); */
// 一次通过好耶 O(∩_∩)O
public class Solution extends VersionControl {
    public int firstBadVersion(int n) {
        int left = 1, right = n;
		while (left < right) {
			int mid = left + (right - left) / 2;
			if(isBadVersion(mid)){ // [!code error]
                if(isBadVersion(mid-1)){ // [!code error]
                    right=mid;
                    mid= left + (right - left) / 2;
                }else{
                    return mid;
                }
            }else{
                 if(isBadVersion(mid+1)){ // [!code error]
                    return mid +1;
                }else{
                    left=mid;
                    mid= left + (right - left) / 2;
                }
            }
		}
        return n;
    }
}
```

::: warning 总结反思
相比与官方解法我每次循环都多调用了两次 isBadVersion 方法,主要是对于循环结束的原因想太多,对二分查找思想领悟的还是不够透彻,造成了冗余调用。

::: details 官方思路
假如第一个选取到的版本结果为true,那么后边版本全抛弃 那么我们继续向前二分查找, 下一次查找结果若为false 则前边全抛弃,直到left==right 结束循环

``` java
public class Solution extends VersionControl {
    public int firstBadVersion(int n) {
        int left = 1, right = n;
        while (left < right) { // 循环直至区间左右端点相同
            int mid = left + (right - left) / 2; // 防止计算时溢出
            if (isBadVersion(mid)) {
                right = mid; // 答案在区间 [left, mid] 中
            } else {
                left = mid + 1; // 答案在区间 [mid+1, right] 中
            }
        }
        // 此时有 left == right，区间缩为一个点，即为答案
        return left;
    }
}
```
