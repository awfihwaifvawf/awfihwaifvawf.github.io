栈

约 798 字大约 3 分钟

2026-04-27

栈（Stack）：也叫做「堆栈」，一种线性表数据结构，只允许在表的一端进行插入和删除操作。

核心特性

栈的操作遵循 后进先出（LIFO） 的原则：

  • 最后放入栈的元素，最先被取出。
  • 就像叠盘子，最后放上去的盘子，总是最先被拿走。

基本操作

栈的常见操作有：

  • 入栈（Push）：在栈顶加入一个新元素。
  • 出栈（Pop）：移除并返回栈顶的元素。
  • 查看栈顶（Peek）：只查看栈顶元素，但不移除。

栈的实现方式

与线性表类似，栈常见的存储方式有两种：「顺序栈」 和 「链式栈」。

  • 顺序栈：采用一段连续的存储空间（如数组）依次存放从栈底到栈顶的元素，并通过指针 toptop 标记当前栈顶元素在数组中的位置。
  • 链式栈：采用单链表实现，每次新元素都插入到链表头部，toptop 始终指向链表的头节点，即栈顶元素的位置。

两种实现方式对比

特性顺序栈链式栈
空间利用率固定大小，可能浪费按需分配，无浪费
扩容操作需要重新分配空间无需扩容
内存碎片较少可能产生碎片
实现复杂度简单相对复杂

顺序栈（数组实现）

时间复杂度：入栈、出栈、查看栈顶均为 O(1)

栈的最常见实现方式是利用数组来构建顺序存储结构。 在Java中ArrayDeque 就是底层数据结构使用循环数组的顺序栈

循环数组（Circular Array / Ring Buffer）的核心是：用两个指针（或一个指针 + 长度）和一个固定大小的数组，通过取模运算让下标在到达数组末尾后“绕回”开头。

  • ArrayDeque结构
    // 存储元素的数组
    transient Object[] elements;

	// 队头指针（包含元素）
    transient int head;

  	// 队尾指针（不包含元素，是空位）
    transient int tail; 

    // 长度限制 扩容后数组最大长度 Integer.MAX_VALUE - 8 
    private static final int MAX_ARRAY_SIZE = ArraysSupport.SOFT_MAX_ARRAY_LENGTH;
  • 入栈操作(好复杂，不想看 QAQ)
  • 出栈操作(好复杂，不想看 QAQ)

链式栈（链表实现）

顺序栈在存储空间上存在一定局限性：当栈满或需要扩容时，往往需要移动大量元素，效率较低。为了解决这一问题，可以采用链式存储结构实现栈。

在java中LinkedList 是 Deque 接口的链表实现，属于链式栈。链表头尾操作均为 O(1)。它允许 null 元素，而 ArrayDeque 不允许
